# Data Science Classification Project  

##  Project Overview  
This project explores a dataset by analyzing its features, visualizing relationships, and applying various machine learning classification algorithms. The goal was to understand which features are most important for prediction and to compare the performance of different models.  

---

##  Key Steps  

1. **Exploratory Data Analysis (EDA)**  
   - Used **Matplotlib** and **Plotly** to visualize feature distributions and relationships.  
   - Identified patterns and potential important features for classification.  

2. **Data Preprocessing**  
   - Handled missing values and outliers.  
   - Normalized/standardized features as needed.  
   - Encoded categorical variables (if any).  

3. **Modeling**  
   - Trained and evaluated multiple classification algorithms, including:  
     - Logistic Regression  
     - Decision Trees  
     - Random Forests  
     - Support Vector Machines (SVM)  
     - K-Nearest Neighbors (KNN)  
   - Used accuracy, precision, recall, and F1-score for evaluation.  

4. **Results & Insights**  
   - Compared models to identify which algorithm performed best.  
   - Analyzed feature importance to understand which attributes most influenced predictions.  

---

##  Tech Stack  

- **Languages**: Python  
- **Libraries**:  
  - Data Analysis: `pandas`, `numpy`  
  - Visualization: `matplotlib`, `plotly`, `seaborn`  
  - Machine Learning: `scikit-learn`
